# Download information for all available series

Download information for all available series

## Usage

``` r
cbrt_meta(token = NULL)
```

## Arguments

- token:

  API Key

## Value

A data frame with metadata for all available series

## Examples

``` r
# Download information for all available series
if (FALSE) cbrt_meta(token = Sys.getenv("EVDS_TOKEN")) # \dontrun{}
```
