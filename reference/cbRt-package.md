# cbRt: R Interface to EDDS of CBRT

An R interface to the Electronic Data Delivery System (EVDS) of the
Central Bank of the Republic of Turkey (CBRT). Provides functions to
retrieve economic data and metadata from the EVDS API v3, with automatic
handling of the 150 observation limit through intelligent chunking based
on series frequency detection.
